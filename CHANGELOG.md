# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [0.1.0] - 2026-06-24

### Added
- Core job search engine with scam filtering and fit scoring (`search.py`)
- Resume and cover letter customization module (`customizer.py`)
- Job application tracker with CSV support (`tracker.py`)
- Browser automation for auto-apply via Playwright (`apply.py`)
- Gmail and Calendar integration module (`integrations.py`)
- Mock test environment for safe verification (`mock_env.py`)
- CLI entry point (`cli.py`)
- Test suite: R1 search, R2 customization, R4 auto-apply tests
- Project architecture document (`PROJECT.md`)
- E2E test infrastructure design (`TEST_INFRA.md`)
